# Waxis Supply Chain & Inventory Automation

Status: product requirements repository

Primary document:

- [PRD.md](./PRD.md)

This repository defines the Waxis Supply Chain & Inventory Automation solution: a visibility, forecasting, replenishment, supplier-risk, warehouse-exception, and AI workflow layer for inventory-heavy business operations.

The PRD is based on Waxis website positioning plus a June 2026 research snapshot across official supply-chain planning, AI-agent, demand-sensing, inventory, and enterprise-governance sources.
